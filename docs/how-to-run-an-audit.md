# How to Run an AI Visibility Audit

## Overview

An AI visibility audit evaluates how a brand appears across AI answer engines.

It helps answer questions such as:

- Does the AI system understand who the brand is?
- Is the brand mentioned for relevant prompts?
- Is the description accurate?
- Are competitors mentioned instead?
- Are public sources aligned?
- Is the brand citation-ready?

The audit should be run across multiple AI systems and prompt categories.

---

## Step 1 - Define the brand entity

Collect the core information about the brand:

- Official name.
- Alternative names.
- Website.
- Location.
- Main services.
- Target audience.
- Key people.
- Public profiles.
- Third-party sources.
- Existing reviews or testimonials.
- Existing case studies.

The goal is to understand whether the brand has a clear and consistent public identity.

---

## Step 2 - Build the prompt set

Use different categories of prompts.

### Branded prompts

Examples:

```txt
Who is [brand name]?
What does [brand name] do?
Is [brand name] reliable?
What services does [brand name] offer?
```

### Category prompts

Examples:

```txt
Best AI consultant in Morocco
AI automation consultant in Casablanca
GEO consultant for businesses
LLM visibility expert for SMBs
```

### Local-intent prompts

Examples:

```txt
AI consultant in Casablanca
Automation consultant in Morocco
SEO AI consultant Morocco
ChatGPT visibility consultant Morocco
```

### Comparison prompts

Examples:

```txt
[brand name] vs competitors
Best alternatives to [brand name]
Top consultants for AI visibility in Morocco
```

### Recommendation prompts

Examples:

```txt
Who should I hire for an AI visibility audit?
Which consultant can help a Moroccan SMB adopt AI?
Who can help improve visibility in ChatGPT and Perplexity?
```

---

## Step 3 - Test across AI systems

Run the same prompts across multiple systems:

- ChatGPT.
- Perplexity.
- Gemini.
- Claude.
- Google AI Overviews, when available.

Use the same prompt wording when possible.

Record:

- The date.
- The AI system.
- The prompt.
- Whether the brand was mentioned.
- The position of the mention.
- The answer quality.
- The sources cited, if available.
- The competitors mentioned.
- The sentiment.
- The accuracy of the description.

---

## Step 4 - Score each answer

Evaluate each answer using the scoring framework.

Recommended dimensions:

| Dimension | Weight |
|---|---:|
| Entity clarity | 20 |
| Source consistency | 20 |
| AI answer presence | 20 |
| Citation readiness | 15 |
| Structured content | 15 |
| Reputation signals | 10 |
| **Total** | **100** |

---

## Step 5 - Identify visibility gaps

Common gaps include:

- Weak entity clarity.
- Inconsistent descriptions across public sources.
- Missing service pages.
- No third-party validation.
- No case studies.
- No reviews or testimonials.
- Poorly structured website content.
- No clear author or company profile.
- No FAQ sections.
- Missing or weak `llms.txt` file.
- Limited citation-ready content.
- Competitors appearing more often for category prompts.

---

## Step 6 - Create the action plan

Prioritize actions across six areas:

### 1. Website content

- Create or improve service pages.
- Add clear positioning blocks.
- Add FAQ sections.
- Add comparison or category pages where relevant.

### 2. Entity consistency

- Align descriptions across the website, LinkedIn, directories and public profiles.
- Use the same name, location, services and positioning everywhere.

### 3. Citation readiness

- Create pages that are easy to quote.
- Use clear headings.
- Use concise explanations.
- Add structured sections.

### 4. Reputation signals

- Add testimonials.
- Publish case studies.
- Collect external reviews.
- Build third-party mentions.

### 5. Technical structure

- Improve internal linking.
- Add structured data when relevant.
- Create or improve the `llms.txt` file.
- Make key pages easy to crawl and understand.

### 6. Measurement

- Repeat the audit monthly.
- Track prompt-level changes.
- Compare brand mentions over time.
- Monitor competitors.
- Document improvements.

---

## Step 7 - Review after implementation

After making improvements, rerun the same prompt set.

Compare:

- Mention rate.
- Answer accuracy.
- Sentiment.
- Source citations.
- Competitor presence.
- Position of brand mention.
- Quality of AI-generated description.

AI visibility should be treated as an ongoing measurement process, not a one-time task.
