# Example Brand Analysis

## Brand

Example Company

## Positioning tested

AI consulting and automation services for SMBs in Casablanca, Morocco.

## Prompt sample

| Engine | Prompt | Result | Notes |
|---|---|---|---|
| ChatGPT | Who is Example Company? | Mentioned | Description was accurate but generic. |
| Perplexity | Best AI consultant in Morocco | Not mentioned | Competitors appeared first. |
| Gemini | AI automation consultant Casablanca | Mentioned | Local positioning was clear. |
| Claude | GEO consultant for SMBs | Not mentioned | Category association was weak. |

## Findings

- The brand is recognized when the company name is used directly.
- The brand is not consistently associated with category-level searches.
- Website and LinkedIn descriptions should use the same service language.
- More third-party references are needed to support recommendation prompts.

## Suggested next steps

1. Add an AI visibility service page.
2. Add a public case study with clear outcomes.
3. Create an `llms.txt` file.
4. Publish a comparison-friendly FAQ.
5. Track the same prompts monthly.
