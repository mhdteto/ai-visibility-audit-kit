## Main issues

1. Weak external reputation signals.
2. Limited visibility in category prompts.
3. Inconsistent service descriptions across sources.
4. Few pages formatted for AI citation.

## Recommended actions

1. Create dedicated service pages.
2. Add FAQ blocks to important pages.
3. Publish 3 case studies.
4. Align LinkedIn, directories and website descriptions.
5. Create an `llms.txt` file.
6. Build third-party mentions and reviews.
